# About SORTIUM

SORTIUM is not a basic sorting visualizer. It transforms algorithm learning into a **real-time immersive experience** with animated bars, glowing highlights, recursion trees, sound frequencies, race mode, step controls, and multiple premium themes.

Designed for students, developers, and anyone who wants to understand sorting algorithms visually.

---

#  Unique Features

##  1. Real-Time Sorting Visualization

Watch array elements sort themselves through animated bars.

* Smooth transitions
* Dynamic bar scaling
* Responsive layout
* High performance rendering

---

##  2. Active Glow Effects

Bars involved in sorting glow live:

*  Comparing bars
*  Swapping bars
*  Pivot element (Quick Sort)
*  Sorted elements

This makes algorithm flow easy to understand instantly.

---

##  3. Recursion Tree Visualizer (Rare Feature)

Visual recursion trees update step-by-step during sorting.

### Merge Sort Tree

* Array keeps dividing into halves
* Shows split phases
* Shows merge phases
* Highlights active range

### Quick Sort Tree

* Shows partitions
* Displays pivot selections
* Left/right recursive calls
* Tracks active branch

This feature makes recursion intuitive.

---

##  4. Race Mode

Run **two algorithms side-by-side** and compare:

* Speed
* Comparisons
* Swaps
* Step progress

Perfect for learning algorithm efficiency visually.

---

##  5. Intelligent Sound System

Sorting actions generate audio tones.

* Frequency changes based on bar values
* Compare sounds
* Swap sounds
* Completion sound
* Volume control
* Mute toggle

Algorithms become audible.

---

##  6. Full Playback Controls

Users control every step:

* ▶ Start
* ⏸ Pause
* ◀ Previous Step
* ▶ Next Step
* ↺ New Array
* ■ Reset

Perfect for detailed study.

---

##  7. Speed Control

Adjust animation speed in real-time.

From slow educational mode to ultra-fast benchmark mode.

---

##  8. Dynamic Array Size Control

Change number of elements instantly.

* Bars auto-fit screen
* No overflow
* Clean scaling
* Performance optimized
