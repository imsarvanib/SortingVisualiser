**SORT VISUALIZER**

  A premium, FAANG‑level sorting algorithm visualizer with a black/red/pink neon theme, glassmorphism UI, telephone‑style sound feedback, and a precise step engine.

  ## Features
  - **Five algorithms**: Bubble, Selection, Insertion, Merge, Quick
  - **Step‑by‑step execution** with forward/backward navigation
  - **Live metrics**: comparisons, swaps, total steps, current index
  - **Complexity panel** with best/average/worst/space for each algorithm
  - **Web Audio sound effects** (toggleable, volume control)
  - **Responsive design**, neon glow, smooth animations

  ## How to Run
  1. Clone the repository or download `SortingFinal.html`.
  2. Open `SortingFinal.html` in any modern browser (Chrome, Edge, Firefox).
  3. For the best experience, use **Live Server** (VS Code extension) to avoid CORS restrictions with Web Audio.
